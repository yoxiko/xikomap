by yoxiko


yoxiko - самописный сканер портов, написанный по прототипу nmap. программа дает максимальную инфомрацию для начинающих пентестеров, с помощью флага --info можно узнать информацию о службе, которая установленна на конкретном порту. 

проект делался не как целевой инструмент дял пентеста, а для изучения сканирования портов. проект будет обновятся по мере возникновения у меня идей.
